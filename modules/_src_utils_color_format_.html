Const format

  • format(color: RgbColor, opacity?: number): ColorType
    • Defined in src/utils/color/format.ts:48

    将RGB类型的颜色转化成各种形式进行使用

    example
    import { color } from 'onex-utils';
const colorType = color.format([255, 255, 255]);

assets(colorType).toEqual({
  rgb: [255, 255, 255],
  rgbColor: 'rgb(255, 255, 255)',
  rgbaColor: 'rgba(255, 255, 255, 1)',
  intHex: 0xffffff,
  stringHex: '#ffffff',
});

    Parameters

    • color: RgbColor

      RgbColor 需要转化的颜色

    • Optional opacity: number

      透明度，值在0-1之间
